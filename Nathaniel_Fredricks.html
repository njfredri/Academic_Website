Software Engineering Intern
May, 2022 to August, 2022

Md. Arafat Kabir completed a summer internship at Cadence Design Systems, working on the development of a physical verification tool named Pegasus. He worked with the team that develops computational geometry algorithms for the tool (Geometry team). During this internship, he developed an algorithm to simplify complex geometric shapes for faster and more efficient processing (details under NDA). The top skills developed and used through out the internship are,

  • Working in a big production level codebase

  • GNU Debugger (GDB)
  • Python Programming
  • Working in a distributed Linux environment
  • Working with code management tools like Emacs, ctags, perforce, etc.
